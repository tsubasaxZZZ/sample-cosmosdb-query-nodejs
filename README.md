# 事前準備

- 実行するマシンに Node.js をインストールします
- Cosmos DB の API の種類は [SQL] を選択します

# 使い方

1. sample.config.js を config.js にコピーします
2. config.js の各設定を実際の環境に合わせます
3. apps.js の querySpec の部分を修正します
4. コンソールから `node app.js` を実行します
